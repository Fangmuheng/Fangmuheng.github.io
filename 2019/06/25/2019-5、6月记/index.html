2019-5、6月记

发表于 2019-06-25 | 分类于 月记及年终 | | 本文总阅读量 次
字数统计: 870字 | 阅读时长 ≈ 2分钟

大二下5、6月的月记总结。其实也没发生什么，就是觉得应该记一记，用来提醒自己这两个月又什么也没有做 :) 以及大致安排一下暑假的生活（好像有点早）。

0.前言

这两个月几乎要被复习逼疯了，并没有发生什么快乐的事情，总之就是又烦又累，整天丧丧的，没什么令人快乐的事情发生。

1.生活回顾

！！好像还真的什么也没做啊！！

  • 嗯，我的向日葵GG了，我果然还是不适合养花花草草的。说起来我还有点想念小可乐了，唉有点难受，虽然是寿终正寝但还是觉得小可乐陪伴了我一整个高三，陪伴我上大学，陪伴我的妈妈好几年的时光……还是没有办法接受没有它的日子。
  • 书么没怎么看，不过倒是把《安德的游戏》第一部看完了，超级棒！我真的超喜欢这本写虫族的科幻文，和大刘的科幻不一样的感觉，虽然我也一直说要看《2001：太空漫游》也一直没看，图书馆似乎有这本书，但我每次都想不起来。
  • 想在暑假的时候试一试微信小程序的制作，如果有时间的话。暑假的安排还是以学习java为主。嗯然后研究一下怎么把本地工程上传到github上（我真蠢现在都不会QAQ
  • 考完了翻译和数值分析，翻译分出了，还没敢看。数值分析最后一道大题错辽，难受，估计只有80出头了（别更低吧。希望接下来的复变数据结构能稍微好点吧。
  • 存款达到了目标金额了，但是目前还没想好应该用来做点什么有意义的事情，那就暂且存着吧，一直到下一个目标。（用来吃喝玩乐总觉得太过无聊，可惜我也没有什么特别想要的东西。如果说现在有想要的，那大概是一条裙子哈哈哈哈但是看不下什么好看的，暂且放着吧。
  • 不记得从什么时候喜欢看新闻了，倒也不是觉得能知道多少东西，就是觉得能了解一点这个世界上发生的事情还是挺有趣的，不过我觉得当我什么时候能思考这些新闻背后的深层意义的时候，才算是真正“看懂了”新闻吧。目前也就处于只接收信息的状态，毕竟了解的太少太少了。
2.暑期安排

说一说暑假安排吧，尽管每次都像是flag。

  • 学习做饭：炒菜。学做水果捞，学做冰镇饮料。（…做一个快乐的肥宅
  • 学习计算机专业课：计算机组成原理。不知道一个月够不够，但愿够。
  • 学习编程：JAVA中期（图形界面、框架。）（我感觉学不完，慢慢来吧
  • 学习微信小程序制作（希望能做出一个类似于ToDoList的小程序
  • 学习化妆（我又说了一遍，又立了一遍flag….希望真的能学会吧
  • 陪妈妈去旅游一次。（天津？武汉？西安？长沙？
  • 考过科二科三。（科二求一定要过啊！科三也不强求了，学会寒假考也行。
  • 每天变得更可爱一点。（hhhh
